# 설정 가이드

## 관리자 계정 설정

관리자 계정을 설정하려면 다음 단계를 따르세요:

### 1. 회원가입
먼저 웹사이트에서 다음 정보로 회원가입을 진행하세요:
- 아이디: `junsu`
- 비밀번호: `sungo8547!`
- 이메일: 유효한 이메일 주소

### 2. 관리자 권한 부여
서버 디렉토리에서 다음 명령어를 실행하세요:

```bash
cd server
npm run set-admin
```

이 명령어는 `junsu` 계정에 관리자 권한을 부여합니다.

### 3. 확인
- 로그아웃 후 다시 로그인하세요
- 헤더에 "관리자" 버튼이 표시되는지 확인하세요
- `/admin` 페이지에 접근할 수 있는지 확인하세요

## 주요 기능

### 판매자 기능
- 상품 등록, 수정, 삭제
- 상품 상태 변경 (판매중 → 예약중 → 판매완료)
- 채팅으로 구매자와 소통

### 구매자 기능
- 상품 검색 및 조회
- 좋아요 기능
- 채팅으로 판매자와 소통
- 상품 예약하기

### 관리자 기능
- 모든 상품 관리
- 사용자 강퇴/해제
- 채팅 기록 관리
- 예약 내역 관리

## 환경 변수

### Server (.env)
```
MONGO_URI=mongodb+srv://...
JWT_SECRET=your_secret_key
PUBLIC_BASE_URL=https://your-domain.com
PORT=4000
```

### Client (.env)
```
VITE_API_BASE_URL=https://your-api-domain.com
```

## 배포

### Railway (Backend)
1. Railway 프로젝트 생성
2. MongoDB 연결 설정
3. 환경 변수 설정
4. `railway up` 또는 GitHub 연동으로 배포

### Vercel (Frontend)
1. Vercel 프로젝트 생성
2. `client` 디렉토리를 루트로 설정
3. 환경 변수 설정 (`VITE_API_BASE_URL`)
4. 배포

## 문제 해결

### 관리자 버튼이 보이지 않는 경우
1. 로그아웃 후 다시 로그인
2. 브라우저 캐시 삭제
3. `npm run set-admin` 명령어가 성공적으로 실행되었는지 확인

### 예약 기능이 작동하지 않는 경우
1. 판매자 본인은 자신의 상품을 예약할 수 없습니다
2. 이미 예약중이거나 판매완료된 상품은 예약할 수 없습니다
3. 로그인이 필요합니다
