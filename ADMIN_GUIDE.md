# 관리자 가이드

## 로그인

관리자 계정으로 로그인하세요:
- **아이디**: `junsu`
- **비밀번호**: `sungo8547!`

로그인하면 자동으로 관리자 권한이 부여됩니다.

## 관리자 페이지 접근

1. 로그인 후 헤더에서 "관리자" 버튼 클릭
2. 또는 직접 `/admin` 경로로 이동

## 관리 기능

### 1. 상품 관리
- 모든 상품 목록 조회
- 상품 정보 확인 (제목, 가격, 판매자, 상태, 등록일)
- 부적절한 상품 삭제

### 2. 사용자 관리
- 모든 사용자 목록 조회
- 사용자 정보 확인 (아이디, 이메일, 이름, 가입일)
- 사용자 강퇴/해제
- 관리자 계정 표시

### 3. 채팅 관리
- 모든 채팅방 목록 조회
- 채팅 참여자 및 상품 정보 확인
- 부적절한 채팅 삭제

### 4. 예약 관리
- 모든 예약 내역 조회
- 예약 정보 확인 (상품, 구매자, 판매자, 상태, 장소)
- 예약 삭제

## 주의사항

- 삭제된 데이터는 복구할 수 없습니다
- 사용자를 강퇴하면 해당 사용자는 로그인할 수 없습니다
- 채팅을 삭제하면 모든 메시지가 함께 삭제됩니다

## 문제 해결

### 관리자 버튼이 보이지 않는 경우
1. 로그아웃 후 다시 로그인
2. 브라우저 캐시 삭제
3. 서버 로그 확인 (콘솔에 "✅ junsu 관리자 로그인 성공" 메시지 확인)

### 관리자 페이지 접근이 거부되는 경우
1. 서버 로그에서 "🔍 관리자 체크" 메시지 확인
2. `isAdmin: true`가 표시되는지 확인
3. 문제가 지속되면 서버 재시작

## 기술 정보

### 자동 권한 부여
- `junsu` 계정은 로그인 시 자동으로 관리자 권한 획득
- 데이터베이스에 `isAdmin: true` 플래그 설정
- 관리자 API 접근 시 자동으로 권한 확인 및 부여

### 보안
- 관리자 API는 인증된 사용자만 접근 가능
- `junsu` 계정 외에는 관리자 권한 부여 불가
- 모든 관리 작업은 로그에 기록됨
